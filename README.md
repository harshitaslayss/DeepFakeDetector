Deepfake Detection using ResNet18

This project detects AI-generated (deepfake) videos using a ResNet18-based CNN architecture trained on the FaceForensics++ dataset.

---

Repository Structure

- `DeepFake.ipynb` → Original experimental notebook (older version)
- `df.ipynb` → Clean, organized final notebook (recommended)
- `cases/` → Supporting scripts, assets, or reference files

---

Model Overview
- Framework: PyTorch
- Backbone: ResNet18
- Dataset: FaceForensics++
- Task: Binary classification (real vs fake)
